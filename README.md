# ⚡ Energy Consumption Analysis

## MCA Minor Project

### 📖 Project Description

This project analyzes household electricity consumption using Python and Data Science techniques. The project performs data cleaning, visualization, anomaly detection, and forecasting to understand electricity usage patterns.

The analysis is carried out using the Individual Household Electric Power Consumption dataset. Different visualization techniques and Machine Learning algorithms are used to identify trends and unusual energy consumption patterns.

---

# 🎯 Objectives

- Analyze household energy consumption.
- Clean and preprocess the dataset.
- Visualize energy consumption using different graphs.
- Detect abnormal power usage using Isolation Forest.
- Forecast future energy consumption using ARIMA.
- Understand consumption patterns for better energy management.

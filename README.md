# Energy Consumption Analysis

## Project Overview
This is an MCA Minor Project developed using Python for analyzing household energy consumption data.

## Objective
- Analyze household electricity usage.
- Visualize energy consumption using different graphs.
- Detect anomalies using the Isolation Forest algorithm.
- Forecast future energy consumption using ARIMA.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Statsmodels

## Dataset
Individual Household Electric Power Consumption Dataset (UCI Machine Learning Repository)

## Graphs Included
1. Energy Consumption Trend
2. Average Energy Consumption by Hour
3. Distribution of Power Consumption
4. Monthly Energy Consumption Share
5. Anomaly Detection Result using Isolation Forest
